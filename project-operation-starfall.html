Overview

Operation Starfall is a first-person shooter set in an orbital space station where players must defend against an invading force bent on sabotaging the station’s systems. Using a complex movement system, futuristic weaponry, and multi-objective gameplay, players must fight off waves of enemies, uncover hidden objects, and fight their way to the control room to stop the invaders.

What I Worked On

Gameplay Systems

  • Built modular weapon systems and implemented multiple weapon types, including heavy gunner
  • Developed seeking projectiles for both enemy and player use
  • Implemented core gameplay features including grapple mechanics and skill systems (sprint)
  • Designed interactive systems such as breakable objects, turrets, and inventory improvements

UI / UX

  • Built UI for a consistent, polished visual style across the game
  • Developed upgrade system UI, boss health displays, and failure state menus
  • Created modular menu/tab systems with improved navigation and usability
  • Optimized UI scaling and resolved layout and asset inconsistencies

Level Design & World Building

  • Designed and completed levels 1–4 from blockout to final polish
  • Built prefab systems to streamline level creation and iteration
  • Implemented gameplay objectives, tutorial triggers, and environmental storytelling
  • Improved navigation and level flow through layout and NavMesh fixes

AI Systems

  • Developed enemy AI with target-seeking behavior
  • Implemented pathing systems for controlled AI movement
  • Integrated AI with combat and objective systems

Bug Fixing & Optimization

  • Resolved a wide range of gameplay, UI, and system-level bugs
  • Fixed visual and functional issues including VFX, UI, and interactions
  • Improved overall stability and polish through continuous testing and iteration
